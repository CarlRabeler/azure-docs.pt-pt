# <a name="azure-network-watcher-documentation"></a>Documentação do Observador de Rede do Azure

Aprenda a utilizar o Observador de Rede, um serviço que lhe permite monitorizar e diagnosticar condições ao nível de um cenário de rede. Os tutoriais, as referências de API e outra documentação mostram como obter informações acerca da sua Rede do Azure com ferramentas como a captura do pacote, a topologia e muito mais.
