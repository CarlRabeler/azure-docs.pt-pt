# <a name="azure-advisor-documentation"></a>Documentação do Azure Advisor

O Azure Advisor é um consultor na cloud personalizado que o ajuda a seguir as melhores práticas para otimizar as suas implementações do Azure. Analisa a configuração do recurso e a telemetria de utilização. Depois, recomenda soluções para ajudar a melhorar o desempenho, a segurança e a elevada disponibilidade dos seus recursos, procurando, ao mesmo tempo, oportunidades para reduzir os seus gastos gerais com o Azure.
