# <a name="azure-active-directory-documentation"></a>Documentação do Azure Active Directory

O Azure Active Directory (Azure AD) é o serviço de gestão de identidades e diretórios baseado na nuvem de multi-inquilino da Microsoft.  O Azure AD oferece funcionalidades de identidade e acesso para aplicações em execução no Azure, bem como no local.
