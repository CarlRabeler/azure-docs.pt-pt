#<a name="cdn-documentation"></a>Documentação do CDN

A Rede de Entrega de Conteúdos do Azure (CDN) fornece aos programadores uma solução global para o fornecimento de conteúdo de largura de banda elevada que está alojada no Azure ou em qualquer outra localização.  Com o CDN pode colocar em cache objetos disponíveis carregados publicamente a partir do armazenamento de blobs do Azure, uma aplicação Web, máquina virtual, pasta de aplicação ou outra localização de HTTP/HTTPS.  O CDN é normalmente utilizado para fornecer conteúdo estático, como imagens, folhas de estilo, documentos, ficheiros, scripts do lado do cliente e páginas HTML.
