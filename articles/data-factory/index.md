# <a name="azure-data-factory-documentation"></a>Documentação do Azure Data Factory

Saiba como utilizar o Data Factory, um serviço de integração de dados na cloud para compor o armazenamento de dados, a movimentação e serviços de processamento em pipelines de dados automatizados. Tutoriais e outra documentação mostram-lhe como configurar e gerir pipelines de dados, bem como mover e transformar dados para análise.
