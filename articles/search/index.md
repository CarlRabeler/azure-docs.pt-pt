# <a name="azure-search-documentation"></a>Documentação do Azure Search

O Azure Search é uma solução de pesquisa como serviço na nuvem que delega a gestão de infraestruturas e de servidores à Microsoft, deixando-o com um serviço pronto a utilizar que poderá preencher com os seus dados e, em seguida, utilizar para adicionar pesquisa à sua aplicação Web ou móvel. O Azure Search permite-lhe adicionar facilmente uma experiência de pesquisa robusta para as aplicações com uma API REST simples ou SDK .NET, sem gerir a infraestrutura de pesquisa ou tornar-se um perito em pesquisa.
